Merci pour votre confiance :

Vous avez choisi le template "portfolio by bootstrap", ce magnifique template va vous permettre de créer votre site web en quelques minutes seulement et sans connaissances particuliéres. Une fois votre template téléchargé ouvrez le dossier "sg.zip" vous retrouverez à l'intérieur plusieurs documents avec parfois des sous-documents (pas de panique). Le fichier "index.html" représente la page d'accueil, le dossier css regroupe le css da la votre page web, "img" regroupe l'ensemble des images nécessaires pour la page web, le dossier "font-awesome" permet de faire télécharger la police au nigateur de votre visiteur, enfin le dossier js regroupe l'ensemble des pages pour faire fonctionner les animations de votre site
"PS: Les textes ou images que vous retrouverez sont purement fictifs vous pouvez les changer à votre guise comme bon vous semble :)"
