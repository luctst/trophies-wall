Merci pour votre confiance :

Vous avez choisi le template "Portfolio", ce magnifique template va vous permettre de créer votre site web en quelques minutes seulement cependant si vous souhaitez modifier de maniére plus complexe ce template quelques notions de html et css seront nécessaires. Une fois votre template téléchargé ouvrez le dossier "pp.zip" vous retrouverez à l'intérieur plusieurs documents avec parfois des sous-documents (pas de panique). Le fichier "index.html" représente la page d'accueil, le css de ce site web se trouve dans le dossier css sous le nom de "creative.css", le dossier font ne recquiert pas votre intervention il contient simplement la police de votre site, le dossier img regroupe toutes les images nécessaires. Et pour finir le dossier JS regroupe tous les fichiers nécessaires pour les animations de votre site vous n'êtes pas obligés de le modifier cependant si vous souhaitez le modelez à votre facon vous aurez besoin de notions en "javascript".
"PS: Les textes ou images que vous retrouverez sont purement fictifs vous pouvez les changer à votre guise comme bon vous semble :)"
